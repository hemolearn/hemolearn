    API Documentation¶

    Sparse Low Rank Decomposition Analysis¶

    hemolearn.SLRDA(n_atoms, t_r[, …])

    Sparse Low-Rank Deconvolution Analysis (SLRDA) is a method to distangle the neural activation and the haemodynamic contributions in the fMRI data (BOLD signal).

    Vascular atlas¶

    hemolearn.atlas.fetch_atlas_basc_2015([n_scales])

    Fetch the BASC brain atlas given its resolution.

    hemolearn.atlas.fetch_vascular_atlas([sym, …])

    Fetch the Havard-Oxford brain atlas given its resolution.

    Generating synthesis data¶

    hemolearn.simulated_data.simulated_data([…])

    Generate simulated BOLD data with its temporal components z and the corresponding maps u.

    Convolution functions¶

    hemolearn.convolution.make_toeplitz(v, …)

    Make Toeplitz matrix from given kernel to perform valid convolution.
