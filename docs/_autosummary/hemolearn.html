    hemolearn¶

    HemoLearn is a Python module to estimate the Haemodynamic Response Function (HRF) in brain from resting-state or task fMRI data (BOLD signal). It relies on a Sparse Low-Rank Deconvolution Analysis (SLRDA) to distangles the neurovascular coupling from the the neural activity.

    The advantages of SLRDA are:

    • The estimation of the HRF for each voxels of the brain.

    • The decomposition of the neural activity in a set of temporal components and

    its associated spatial map that describe a function network in the brain.

    The disadvantages of SLRDA include:

    • If the temporal resolution in the fMRI data is too low (TR > 2s) it’s likely

    that the analysis will not found major difference between the HRFs. This is due to the fact that the common time-to-peak difference between HRFs within the same brain varies at maximum of 2 s, if the temporal resolution is greater that this, no effect will be estimated.
